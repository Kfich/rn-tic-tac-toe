## Getting Started

#### 1. Unzip and Install

```
# Install expo cli
npm install expo-cli --global

# Unzip tic-tac-toe, Navigate to root folder and Install dependencies
cd tic-tac-toe && yarn install (or npm install)

# Start with expo
cd expo start
```

#### 2. Open tic-tac-toe in your iOS or Android simulator

Once development server running, follow expo prompts to start your app on iOS or android simulator
